# 星光火火_RDKX3智慧医疗巡检机器人
队伍：星光火火队 | 学校：阜阳理工学院 | 主控：RDK X3
## 一、赛题实现
基于OriginCar Pro底盘，完成3大赛事任务：
1.大厅自主避障行驶，二维码识别播报任务；
2.诊疗区黄色通道环形巡检，人形立牌VLM图生文语音屏显输出；
3.自主返回发车P点，全程无人工干预，限时180秒。
## 二、硬件配置
原厂底盘电机舵机不变，搭载RDK X3、MIPI相机、激光雷达、语音OLED屏。
## 三、软件架构
ROS分层设计：底盘驱动、多传感器融合、二维码检测、轻量化多模态大模型推理、状态机路径规划。
## 四、罚时优化
激光提前避障减少碰撞，视觉实时修正通道偏移，AI低延迟识别规避图文播报罚时。
## 五、仓库内容
包含全套源码、RKNN模型、硬件接线图纸、赛道调试日志、一键启动脚本。
